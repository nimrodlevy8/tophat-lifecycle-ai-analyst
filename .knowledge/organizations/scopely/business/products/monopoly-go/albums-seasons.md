# Monopoly Go — Albums & Seasons

## Season Structure

- Each season lasts ~**50–65 days** (~2 months)
- Themed, often IP-based: "Hello Kitty," "Harry Potter," "Cozy Comforts," "Bon Appetit," "Speed Racers"
- IP seasons inflate reactivations and LP spending; following non-IP season may look like a decline but is **baseline reversion**
- Season composition tracked: Solo vs. Social minigame day mix (e.g., "22d/17d Solo/Social")
- Compare at same day count (e.g., "First 39 Days") for normalization

## Album Sets & Completion

1. Each season introduces a new album with multiple themed sets
2. Each set: ~6–9 stickers
3. Typically 10–15 sets per album
4. Each sticker has a rarity rating (1–5 stars)
5. Completing a set → set completion bonus
6. Completing ALL sets → grand album completion prize (massive rolls, exclusive items)

## Sticker Rarity Tiers

| Rarity | Stars | Drop Rate | Notes |
|--------|-------|-----------|-------|
| Common | 1★ | ~40–50% | Easy to collect, frequently duplicated |
| Uncommon | 2★ | ~25–30% | Standard piece |
| Rare | 3★ | ~15–20% | Takes more packs |
| Epic | 4★ | ~5–8% | Significant milestone |
| Legendary | 5★ | ~1–3% | Album-completion gate; hardest to find |

**Golden Stickers:** Untradeable variants (often 5-star). Prestige/collection items and completion gates. Some only from specific events or purchases.

## Trading Mechanics

| Rule | Detail |
|------|--------|
| Daily send limit | ~5 stickers per day |
| Tradeable | Only duplicates |
| Untradeable | Golden stickers (usually), 5-star (rules vary by season), stickers within first ~48h of release |
| Rarity restrictions | High-rarity often trade-locked or same-tier-only (4★ for 4★) |
| Friends only | Must be on friends list |

## Retention & Monetization Drivers

- Album completion as a 2-month goal drives daily play
- Set completion creates intermediate milestones
- Trading creates social bonds and daily engagement
- "Last few stickers" urgency as season approaches end
- FOMO: incomplete albums lost at season end
