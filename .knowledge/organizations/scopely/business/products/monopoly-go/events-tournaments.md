# Monopoly Go — Tournaments & Milestones

## Tournaments

### How They Work
- Run continuously — almost always an active tournament overlapping with minigame events
- Scoring: points earned from rolling dice, landing on spaces, building landmarks, attacks/heists
- Leaderboard: brackets of ~50–100 players matched by level/activity
- Duration: rounds typically 30 min to 2 hours, multiple rounds per day

### Tournament Types

| Type | Description |
|------|-------------|
| Standard | General scoring from combined gameplay |
| Build | Bonus points for building/upgrading landmarks |
| Rent | Bonus points for collecting rent |
| Attack/Heist | Bonus points for Smashdown and Bank Heist |
| Rush/Flags | Time-limited (10–30 min) with 2x–3x multipliers. Flag pickups on board for bonus points. |

### Reward Structure
Based on leaderboard placement (top 1, top 3, top 5, top 10, top 50%, participation).

**Critically: tournament rewards include minigame event tokens — the primary source of minigame currencies.**

## Milestones

1. Run alongside tournaments as a separate cumulative reward track
2. Track cumulative actions across the entire event period (do NOT reset between rounds)
3. Series of ascending point thresholds (~20–40+ nodes)
4. Rewards escalate: early = small cash/rolls/1 token → late = large rolls/rare stickers/significant tokens → final = massive rewards

## Flag/Rush Events

- **Flags:** Special board pickups during rush events; landing on/passing collects for bonus tournament points. Appear for 10–30 minutes.
- **Rush Events:** Short high-intensity windows (10–30 min) with 2x–3x multipliers. Push notifications alert players. Community strategy: save rolls for these.

## Why Tournaments/Milestones Matter

This is the critical economy link:

```
Rolls → Board Rolling → Tournament Points → Minigame Tokens
```

Without tournament/milestone participation, players cannot effectively earn minigame tokens. Tournaments are the **engine powering the entire minigame economy**.

Tournament participation is upstream of all minigame engagement — any analysis of minigame performance must consider tournament behavior as a confounding or mediating variable.
