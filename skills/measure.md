# Measurement Framework

## When to use
Design KPIs, success metrics, or instrumentation for a feature or initiative.

## Methodology
1. Clarify the initiative's goal and expected player behavior change.
2. Define primary metric (the one number that says "this worked").
3. Define secondary metrics (supporting evidence, guardrails).
4. For each metric:
   - Precise definition (numerator/denominator, inclusion criteria)
   - Data source and feasibility
   - Expected baseline and minimum detectable effect
5. Identify guardrail metrics (things that shouldn't get worse).
6. Recommend measurement approach: pre/post, AB test, cohort comparison.
7. Define success criteria: what threshold = ship decision?
