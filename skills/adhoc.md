# Ad Hoc Analysis

## When to use
Stakeholder asks a specific question needing a fast, accurate answer.

## Methodology
1. Clarify the question — restate it to confirm understanding.
2. Identify the relevant tables and metrics from context/schemas/.
3. Write a query that answers the question directly.
4. Present results with:
   - The answer (one sentence)
   - Supporting data (table or chart description)
   - Segment breakdowns if relevant
   - Recommendation or next question to ask
