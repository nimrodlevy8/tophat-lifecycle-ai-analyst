# Segmentation Framework

## New Users
Segment by:
- `geo_tier`
- `country_name`
- `platform`
- `publisher_name`
- `channel`

## Reactivations
Segment by:
- `geo_tier`
- `country_name`
- `platform`
- Days since last activity (bins)
- Number of times reactivated in lifetime
- Board level when reactivating
- Lifetime activity days when reactivating
- Customer / non-customer
- LTV when reactivating
- RTUE mini game
